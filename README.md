# createmdlink
Create markdown link.

# Dependency
* bash
* curl

# Prepare

Copy `createmdlink` on your PATH

```
# cp createmdlink /usr/local/bin/
# which createmdlink 
/usr/local/bin/createmdlink
```

# Example

```
# createmdlink https://github.com/chaspy/createmdlink

```
